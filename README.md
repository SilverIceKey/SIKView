# SIKView

SIKView是一个放置自定义View的仓库，本仓库放置一些自定View和一些预设资源，用于复制使用。

#### 目前存在以下自定义View：

- [RemoteControlView](./app/src/main/java/com/sik/view/RemoteControlView.kt)类似遥控的控制器的自定义View
